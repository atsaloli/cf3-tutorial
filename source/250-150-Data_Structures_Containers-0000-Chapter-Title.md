### Data Containers

**Data container**
: A data container is a lot like a JSON document, it can be a key-value
map or an array or anything else allowed by the JSON standard with
unlimited nesting.

Example:

```json
{
  "Pizza": "Pepperoni",
  "Cities": [
    "London",
    "Paris",
    "Rome"
  ],
  "Games": {
    "Nintendo": [
      "Mario Bros",
      "Contra",
      "Zelda"
    ],
  }
}
```
